# Météo des aéroports

To build all files in bin/ :

```shell
make build
```

To run temperature file :

```shell
make temperature
```

## Composants

Protocoles utilisés : MQTT & HTTP

Database NoSQL Key-Value : REDIS (ou autre base NoSQL)

Broker MQTT : Moquitto (ou autre broker)
